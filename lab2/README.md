### Процесс сборки

1. Откройте командную строку в корневой директории проекта
2. Запустите скрипт сборки:
   ```
   build.bat
   ```


## Запуск сервера

После успешной сборки сервер можно запустить несколькими способами:

### Использование скрипта запуска

Из корневой директории проекта выполните:
```
bin\run_server.bat
bin\run_client.bat
```
